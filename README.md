# Nexus Digital - Sistema de Ventas

Sistema completo de gestión de ventas para Nexus Digital.

## Características

- Gestión de usuarios (Admin/Vendedor)
- Control de inventario
- Registro de ventas
- Gestión de clientes
- Reportes y estadísticas
- Diseño responsive

## Tecnologías

- PHP 7.4+
- MySQL
- HTML5, CSS3, JavaScript
- Font Awesome 6

## Instalación

1. Clonar el repositorio
2. Configurar base de datos
3. Configurar conexión en `/config/conexion.php`
4. Iniciar sesión con admin/admin123

## Credenciales por defecto

- **Admin:** admin / admin123
- **Vendedor:** vendedor / vendedor123
